# Editors

My favorite editors and viewers:

- QLMarkdown
  - a macOS Quick Look extension for viewing Markdown files
  - `brew install --cask qlmarkdown`
- Logseq
  - help organize your thoughts and ideas
  - download from Logseq site
- JetBrains IntelliJ IDEA
  - for not only editing Markdown files but Java development
  - download from JetBrains site
